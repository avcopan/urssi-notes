# Git

Concepts:
 - Working Directory (WD)
 - Staging Area (SA)
 - Commit History (CH)

Commands:
```
git init    # Initialize the directory with a .git/ folder
git status  # See what git is thinking
git add <file/folder name>  # Adds file from WD to SA
git commit                  # Moves file from SA to CH
git log                     # Show my CH
```
